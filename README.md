# Nested-JSON-Analysis
Nested JSON from External API and Analysis

## Requirements

* Python = 3.7
* everything in Conda environment.yml file

## Installation 

Install Miniconda, then:
~~~
conda env update
conda activate nested_json_env
~~~

## Run it
~~~
python driver_scripts/main_json_analysis.py
~~~
